Если **System Code** — это «что», а **Info Code** — это «действие», то **Info Code Variant** (вариант информационного кода) — это **нюанс** или **способ** выполнения этого действия.

Простыми словами: это одна и та же операция (например, «ремонт»), но выполняемая разными способами или в разных условиях.

---

### 1. Зачем нужен этот символ?
Бывают ситуации, когда одну и ту же деталь можно обслужить по-разному. Например, один способ требует специального стенда, а другой можно выполнить прямо «в поле» подручными средствами. Чтобы не путать эти две разные инструкции, им присваивают разные буквы варианта.

* **Буква «A»** — это почти всегда **стандартный** вариант (основной способ).
* **Буквы «B», «C» и далее** — это альтернативные способы.

---

### 2. Примеры из жизни
Давайте возьмем наш пример с колесом.
Система: **W10** (Колесо)
Инфо-код: **520** (Установка)

Теперь добавим **Info Code Variant**:

* **W10 - 520 - A**: Установка колеса с помощью автоматического гидравлического подъемника (стандарт для сервисного центра).
* **W10 - 520 - B**: Установка колеса вручную с помощью домкрата (вариант для аварийной замены на дороге).

Хотя цель одна и та же (поставить колесо), шаги в инструкции будут абсолютно разными. Буква в конце помогает системе хранения данных (CSDB) разделять эти инструкции.

---

### 3. Где он стоит в коде модуля (DMC)?
В полной строке кода модуля данных этот параметр идет сразу после информационного кода.

`MYSYSTEM-A-D00-00-00-00A-520A-A`
^
**Вот он**

### 4. Основные правила
* **Регистр:** Всегда используется заглавная латинская буква.
* **Алфавит:** Обычно идут по порядку (A, B, C...).
* **Исключения:** Буквы **I** и **O** часто пропускают, чтобы их не путали с единицей и нулем.

---

### Резюме
**Info Code Variant** — это уточнение. Если вы видите букву **«A»**, значит перед вами стандартная процедура. Если стоит **«B»** или **«C»** — это какой-то альтернативный метод выполнения той же самой работы.

Все три параметра вместе создают уникальный «ключ» к знанию:
1.  **System Code (D00):** Двигатель.
2.  **Info Code (520):** Установить.
3.  **Variant (B):** Вручную (альтернативный способ).
